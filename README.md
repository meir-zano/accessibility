# Accessibility Widget ♿

ספריית JavaScript קלה ופשוטה שהכנתי להוספת תפריט נגישות לאתרים. המטרה הייתה ליצור כלי שלא מעמיס על האתר, בלי תלויות חיצוניות (Zero Dependencies), שעובד חלק ופשוט "רק עובד".

## 🚀 הטמעה מהירה

פשוט תדביק את השורה הזו לפני סגירת תגית ה-`</body>` באתר שלך:

```html
<script src="https://cdn.jsdelivr.net/gh/meir-zano/accessibility@latest/main.min.js"></script>
```

---

## ⚙️ התאמה אישית (Configuration)
העדכון החדש מאפשר לך לשלוט בנראות ובמיקום של הווידג'ט בלי לגעת בקוד המקור שלי. פשוט תגדיר את האובייקט `mzAccessibilityConfig` **לפני** טעינת הסקריפט:

```html
<script>
  window.mzAccessibilityConfig = {
    theme: 'modern',         // עיצוב: 'modern', 'sidebar', 'dark', 'compact'
    position: 'bottom-right',// מיקום: 'bottom-right', 'bottom-left', 'top-right', 'top-left', 'mid-right', 'mid-left'
    primaryColor: '#0D6EFD', // צבע המותג שלך
    defaultLang: 'he',       // 'he' או 'en'
    maxTextZoomPercent: 200  // מקסימום הגדלת טקסט
  };
</script>
```

### אפשרויות הגדרה נוספות:
* **theme:**
    * `modern`: ברירת המחדל, עיצוב צף ונעים.
    * `sidebar`: סרגל צד שנפתח על כל הגובה של המסך.
    * `dark`: מראה כהה ומינימליסטי.
    * `compact`: עיצוב רשימה (טור אחד) לחיסכון במקום.
* **position:** ניתן לשים את האייקון בכל פינה באתר (`bottom-right`, `top-left` וכו').
* **primaryColor:** תכניס כאן את הצבע הראשי של האתר שלך, והווידג'ט יתאים את עצמו אוטומטית.

## 🛡️ "אל תיגע לי בעיצוב" (mz-ignore)
יודע איך זה כשסקריפט נגישות פתאום מגדיל לך את הלוגו או הורס אלמנט גרפי ספציפי? לא כיף.
אם יש אזור באתר (או אלמנט בודד) שאתה רוצה שהסקריפט שלי **לא יתקרב אליו**, פשוט תוסיף לו את הקלאס: `class="mz-ignore"`.
הסקריפט יתעלם מכל מה שנמצא בתוך האלמנט הזה.

## ✨ תכונות עיקריות
* **שינוי ניגודיות:** מצב כהה, בהיר וניגודיות גבוהה.
* **התאמת טקסט:** הגדלה/הקטנה (עם מנגנון שמונע מהטקסט להשתגע).
* **שיפור קריאות:** גופן קריא, ריווח טקסט והדגשת קישורים.
* **מצב נוחות:** עצירת אנימציות, העלמת מדיה (סרטונים/תמונות), וזכוכית מגדלת.
* **הקראה קולית:** עובד מצוין עם מנועי הדיבור של הדפדפן.
* **עריכה דינמית:** תמיכה ב-SVG אייקונים (חיים או כ-Background).

## 🛠 טכנולוגיות
* **Vanilla JavaScript:** קוד נקי, מהיר, בלי ספריות כבדות.
* **LocalStorage:** כל ההגדרות נשמרות למשתמש גם אחרי ריענון.
* **Accessibility First:** בנוי עם `tabindex` ו-`role` כדי לעבוד טוב עם מקלדות.
