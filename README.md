
# Accessibility Widget ♿

ספריית JavaScript קלה ופשוטה להוספת תפריט נגישות לאתרי אינטרנט. הכלי מאפשר למשתמשים להתאים את חווית הגלישה לצרכים האישיים שלהם בלחיצת כפתור, בהתאם לתקני הנגישות המקובלים.

## 🚀 הטמעה מהירה (Quick Start)

כדי להוסיף את תפריט הנגישות לאתר שלך, פשוט העתק והדבק את הקוד הבא לפני סגירת תגית ה-`</body>`:

```html
<script src="https://cdn.jsdelivr.net/gh/meir-zano/accessibility@latest/main.min.js"></script>
```

## ✨ תכונות עיקריות (Features)

הווידג'ט כולל מגוון רחב של כלים לשיפור חווית המשתמש:

* **שינוי ניגודיות (Contrast):** מעבר בין מצב כהה, מצב בהיר וניגודיות גבוהה.
* **התאמת טקסט:** הגדלה והקטנה של הגופנים באתר.
* **שיפור קריאות:** שינוי הגופן לסיסטם-פונט קריא יותר (Readable Font).
* **הדגשת קישורים:** הוספת קו תחתון או רקע בולט לכל הקישורים בעמוד.
* **סמן עכבר מוגדל:** הפיכת סמן העכבר לגדול ובולט יותר לניווט קל.
* **ביטול אנימציות:** עצירת אלמנטים זזים שעשויים להפריע לריכוז.

## 🛠 טכנולוגיות

* **Vanilla JavaScript (ES6+)** - ללא תלות בספריות חיצוניות (Zero Dependencies).
* **CSS3** - שימוש ב-Custom Properties לעיצוב נקי ומודרני.
* **jsDelivr** - הגשה מהירה דרך CDN.
